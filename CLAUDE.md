# CLAUDE.md

## Project

`jrgmadrid.dev`: real-face personal landing for the developer at GitHub `jrgmadrid`. Brand register, distinct from the Luxuria Morningstar VTuber surface at `morningstar.moe`. Audience: engineering peers and the technical-cultural community. Primary CTA: convert casual visitors into followers, not contact-form fills.

## Tech

Astro 6 (TS strict) + Tailwind v4 via `@tailwindcss/vite` (CSS-first, no `tailwind.config.js`). Deployed static to Vercel. Node >=22.12.0.

## Design context

Read these before any frontend work. Both files are the source of truth:

- **`PRODUCT.md`** (strategic): brand personality, voice signatures, anti-references, design principles.
- **`DESIGN.md`** (visual, seed mode): Creative North Star "The Late-Tokyo Editorial." Committed night-sky purple (oklch(28–32% 0.08–0.10 285); anchor: riverside frame from *Night Is Short, Walk On Girl*). Editorial display + monospaced body. Restrained motion. Re-run `/impeccable document` once real code exists to switch the seed to scan mode.

## Workflow

- Substantive frontend work routes through the `impeccable` skill (`/impeccable craft|shape|critique|polish`). Not optional.
- Brand register: design IS the product. Use `reference/brand.md`, not `reference/product.md`.
- The **Not-By-Default Rule** is load-bearing. Nothing ships that another Claude session would have generated by reflex. See DESIGN.md Don'ts for the full reflex list (Inter/Geist/Plus Jakarta, shadcn cards, lucide-react, Tailwind default grays, hero-feature-grid-CTA-footer, gradient text, glassmorphism).
- Voice in copy: lowercase by default, internet-native vocabulary coexisting with technical density. See PRODUCT.md "Voice signatures."
- No em dashes. Use commas, colons, semicolons, periods, parentheses.
- Commits: no `Co-Authored-By: Claude` trailer.
